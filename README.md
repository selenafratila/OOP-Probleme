# OOP-Probleme

 O colecție de probleme de algoritmică și design software dezvoltate în C++, axate pe modelarea sistemelor din lumea reală prin paradigmele POO.
## Concepte
 Encapsulation & Data Hiding: Gestionarea atributelor private/protected, utilizarea constructorilor (implicit, inițializare, copiere) și a destructorilor pentru managementul dinamic al memoriei.  
 Operator Overloading: Implementarea operatorilor pentru stream-uri (>>, <<), indexare ([]), apel de funcție (()), operații pe biți (|=) și operatori de comparație pentru a crea cod intuitiv și eficient.  
 Inheritance & Polymorphism: Utilizarea claselor de bază și derivate pentru reutilizarea codului și implementarea metodelor virtual (late binding) pentru comportament dinamic.  
 Dynamic Memory Management: Alocarea și dealocarea manuală a resurselor folosind pointeri (new, delete), esențială pentru optimizarea performanței în sisteme cu resurse limitate.
